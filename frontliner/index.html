How does InstaSent Work?

Understanding the Brain Behind InstaSent

InstaSent operates through the integration of three core layers: data scraping, artificial intelligence integration, and result visualization. The process begins when users input the Instagram username and the number of posts, after which the request is sent to the backend, triggering Selenium to perform automated web scraping.

Raw (unpreprocessed) comment data is then processed using Pandas and classified by a Support Vector Machine (SVM) SVM trained on over 1000+ data samples and map every data into three sentiment categories: positive, neutral, and negative. The analysis results, including total post statistics and sentiment percentages, are then sent back to the frontend.

In the final stage, the data is captured by Vue.js and rendered into visually appealing components with the support of Chart.js.
